# learnSolidity-SmartContract
Solidity SmartContract
 Dasar-dasar Solidity
     -Tipe data: uint, int, bool, address, bytes, string, array, mapping, struct.
     -Variabel: state, lokal, global (msg.sender, block.timestamp).
     -Fungsi: visibilitas (publik, pribadi, internal, eksternal), tampilan, murni, berbayar.
     -Modifier, event, penanganan kesalahan (memerlukan, mengembalikan, menegaskan).
 Lanjutan
     -Inheritance dan multiple inheritance (linearisasi C3).
     -Perpustakaan (SafeMath, meski Solidity 0.8+ sudah built-in overflow check).
     -Antarmuka dan kontrak abstrak.
     -Fallback dan menerima fungsi.
     -Optimasi gas: packing storage, loop minimal, gunakan calldata daripada memory untuk read-only.
